Software Section

Key Performance Area

  1. Meet Project Requirements Project requirements are meet through solid software specification and understanding of both functional and non functional requirements at each given stage.
  2. Deliver in Time Time management is at the core of application development and is achieved through continuous program tracking using various tools such as team velocity charts.
  3. Keep the Quality high Code reviews and continuous testing of application help in achieving Quality control of every development.
  4. Security Secure coding is at the heart of modern application development in order to avoid application vulnerabilities introduced during development and deployment
  5. Innovation Software Development is driven by the desire to eliminate long queues and manual process.
  6. Application Design and Documentation The section is responsible for coming up with components, or systems that meet user requirements and documenting all changes effected.
  7. Database Management Determining was requires to be stored and how it should be store, including creating relationship among the stored data.
  8. Systems Integration Bring together components of different system to work as one.
